# energy

このUnityのシステムは、Chaiさんのサーバーが動作していることが条件となります。
SokcetIOを利用しています。
The System needs Server Program which develpoed by Mr.Chai.This Unity system use SoketIO API.


１、Unityの\SocketClient\Assets\Sceneフォルダにある、"DemoScene2"を開いてください。
First, please open the scene file "DemoScene2" at \SocketClient\Assets\Scene\ directory.


２、HierarcyビューのSocketIOのurlを、IPとPortをサーバーに適した条件に変更して、ビルドしてください。
Second, you should change IP Adress and Port number of url at SocketIO in Hirarcy view.


３、あとはビルドしたexeファイルを実行するだけでOKです！
Finally, you build exe file of unity application and do the exe file.


機能としては、日本の発電量のクラスターを含めた表示と個人の発電量に特化した表示、世界の発電量の表示があります。

scriptの中の構造はぐちゃぐちゃになっていますので、注意！
